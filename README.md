# Spring PetClinic CI/CD Project

This project demonstrates a complete CI/CD pipeline using Jenkins and Docker for a Java-based Spring Boot application.

---

## 📌 Project Overview

Spring PetClinic is a sample Spring Boot application used to demonstrate how DevOps tools can automate the build, test, and deployment process.

This project covers:
- GitHub source code management
- Jenkins pipeline automation
- Docker containerization
- Deployment on a server

---

## 🚀 Features

- Spring Boot Application (Java 17)
- Maven Build Automation
- Docker Image Creation
- Jenkins CI/CD Pipeline
- Automated Deployment
- Browser-based application access

---

## 🛠️ Technologies Used

- Java 17  
- Spring Boot  
- Maven  
- Docker  
- Jenkins  
- GitHub
